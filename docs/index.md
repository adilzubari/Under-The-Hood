# SE Docs

A personal software engineering knowledge base — concepts, deep dives, and **interactive interview prep**.

> **Built for two audiences:** my future self refreshing a topic before an interview, and any dev who wants a clean reference.

## How to use this site

### 🎯 Studying for interviews
Open any topic. Skip to **Interview Questions** at the bottom. Each question is collapsed — read it, try to answer in your head, then click to reveal the model answer. No peeking.

### 📚 Learning a topic for the first time
Read top-to-bottom: **TL;DR → Concept Overview → Deep Dive**. The Deep Dive has runnable Python examples and Mermaid diagrams.

### 🔍 Looking something up
Use the search bar (top-right or press <kbd>/</kbd>). Indexed across every topic.

### 🏗️ Practicing system design
Go to the [Scenarios Bank](scenarios-bank/index.md). Each scenario walks through a realistic design problem (constraints → approach → solution → trade-offs).
